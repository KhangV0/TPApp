## 1. NhaCungUng — Hiển thị Logo & Ảnh trụ sở

### Vấn đề
Ô Logo đơn vị và Ảnh trụ sở bị gán nhầm nhau (ngược chiều) trên cả trang CMS lẫn trang public.

### Thay đổi

**`Views/NhaCungUng/Detail.cshtml`**
- Xóa ảnh trụ sở ra khỏi khu vực header cạnh logo.
- Logo đơn vị (`LogoUrl`) → chỉ hiển thị ở góc trái header.
- Ảnh trụ sở (`ImageUrl`) → chuyển vào Tab 4 "Hồ sơ năng lực số", có thể click để xem ảnh lớn.

**`Controllers/NhaCungUngController.cs`**
- `LogoUrl` ← `entity.Logo` (cột Logo trong DB)
- `ImageUrl` ← `entity.HinhDaiDien` (cột HinhDaiDien trong DB)
- Danh sách Index và sidebar "Nhà cung ứng khác": ưu tiên `x.Logo`, fallback sang `x.HinhDaiDien`.

## 2. NhaCungUng — Hồ sơ năng lực: Video & Chứng nhận

### Vấn đề
Tab "Hồ sơ năng lực số" chỉ hiển thị link text thuần, không hiện video hay ảnh chứng nhận trực tiếp.

### Thay đổi

**`Views/NhaCungUng/Detail.cshtml`**
- Video `.mp4` trực tiếp → render `<video controls>` với nút **Tải video về máy**.
- Video YouTube → embed iframe như cũ.
- Link video khác → nút "Xem video / Tải video".
- Chứng nhận:
  - `.jpg/.png` → render ảnh inline, click xem lớn.
  - `.pdf` → render iframe 700px để đọc ngay trên trang.
  - Luôn kèm nút **Tải file chứng nhận**.

## 3. Export Excel — Quản lý Công nghệ (SanPhamCNTB)

### Mới hoàn toàn

**`Services/IExcelExportService.cs`** *(mới)*
- Interface tái sử dụng: `Export<T>(IEnumerable<T> data, string fileName)`.

**`Services/ExcelExportService.cs`** *(mới)*
- Implementation dùng thư viện **EPPlus 7.6.1**.
- Header xanh dương, tô màu xen kẽ hàng, freeze row 1, auto-fit columns.
- Tự động phát hiện URL (`http://`, `https://`) → render thành **hyperlink có thể click** trong Excel (màu xanh + gạch chân).
- Đọc `[Display(Name="...")]` làm tiêu đề cột, fallback về tên property.

**`Program.cs`**
- Đăng ký DI: `builder.Services.AddScoped<IExcelExportService, ExcelExportService>()`.

**`Areas/Cms/Controllers/SanPhamCNTBController.cs`**
- Inject `IExcelExportService` vào constructor.
- Thêm action `[HttpGet] ExportExcel(...)`:
  - Áp dụng đúng bộ lọc (keyword, statusId, ncuId, xuatXuId, siteId, createdFrom, createdTo, productType).
  - Không phân trang (xuất toàn bộ).
  - Build cột `PublicUrl` = `https://{host}/2-cong-nghe-thiet-bi/{productType}/{slug}-{id}.html`.
  - Trả file `.xlsx` tên: `CongNghe_yyyyMMdd.xlsx`.
- Thêm `using TPApp.Controllers;` để dùng `ProductController.MakeURLFriendly`.
- Thêm property `PublicUrl` vào class `SanPhamCNTBListItem`.

**`Areas/Cms/Views/SanPhamCNTB/_ListLayout.cshtml`**
- Thêm nút **Export Excel** (màu xanh lá outline) vào footer form tìm kiếm, bên phải nút "Đặt lại".

**`Areas/Cms/Views/SanPhamCNTB/CongNghe.cshtml`**
- Thêm JS handler `$('#btnExportExcel').on('click', ...)`: serialize filter hiện tại + `productType=1` → `window.location` tới action `ExportExcel`.

## 📋 Hướng dẫn áp dụng Export Excel cho trang list khác

> Service `IExcelExportService` đã hoàn toàn tái sử dụng được. Chỉ cần làm 3 bước sau cho mỗi module mới.

### Bước 1 — Thêm property `PublicUrl` vào ViewModel (nếu cần URL)

Tìm class ViewModel của trang list (ví dụ `NhaCungUngListItem`, `ChuyenGiaListItem`…) và thêm:

```csharp
public string? PublicUrl { get; set; }
```

Thêm `[Display(Name = "Tên cột")]` vào các property nếu muốn header Excel đẹp hơn:

```csharp
[Display(Name = "Tên đầy đủ")]
public string FullName { get; set; } = "";

[Display(Name = "Link trang chi tiết")]
public string? PublicUrl { get; set; }
```

---

### Bước 2 — Thêm action `ExportExcel` vào Controller

Inject `IExcelExportService` vào constructor (nếu chưa có):

```csharp
private readonly IExcelExportService _excelExport;

public MyController(..., IExcelExportService excelExport)
{
    _excelExport = excelExport;
}
```

Thêm action (copy và điều chỉnh filter cho phù hợp module):

```csharp
[HttpGet]
public async Task<IActionResult> ExportExcel(
    string? keyword, int? statusId,
    DateTime? createdFrom, DateTime? createdTo)
{
    var query = _context.MyEntities.AsNoTracking();

    // Áp dụng đúng bộ filter như action Index
    if (!string.IsNullOrWhiteSpace(keyword))
        query = query.Where(x => x.Name!.Contains(keyword));
    if (statusId.HasValue)
        query = query.Where(x => x.StatusId == statusId.Value);

    query = query.OrderByDescending(x => x.Created);

    var items = await query
        .Select(x => new MyListItem
        {
            Id       = x.Id,
            Name     = x.Name ?? "",
            // ... các trường khác
        })
        .ToListAsync();

    // Build PublicUrl nếu cần
    var baseUrl = $"{Request.Scheme}://{Request.Host}";
    foreach (var item in items)
    {
        var slug = ProductController.MakeURLFriendly(item.Name);
        item.PublicUrl = $"{baseUrl}/slug-trang/{slug}-{item.Id}.html";
    }

    return _excelExport.Export(items, $"TenModule_{DateTime.Now:yyyyMMdd}");
}
```

> ⚠️ Nhớ thêm `using TPApp.Controllers;` ở đầu file nếu dùng `ProductController.MakeURLFriendly`.

### Bước 3 — Thêm nút và JS vào View

**Trong file `.cshtml` của trang list**, thêm nút vào footer form tìm kiếm:

```html
<button type="button" class="btn btn-outline-success btn-sm" id="btnExportExcel"
        title="Xuất danh sách hiện tại ra Excel">
    <i class="fas fa-file-excel me-1"></i> Export Excel
</button>
```

**Trong `@section Scripts`**, thêm JS handler:

```javascript
$('#btnExportExcel').on('click', function () {
    // Thay 'ExportExcel' và 'TenController' cho đúng module
    var params = $('#searchForm').serialize();
    window.location = '@Url.Action("ExportExcel", "TenController", new { area = "Cms" })?' + params;
});
```
